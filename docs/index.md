## OpenBSD Homelab

Simple homelab setup using OpenBSD and ansible.

### Available applications

* [Calibre Web](calibre_web)
* [FreshRSS](freshrss)
* [Hashicorp Vault](vault)
* [Lego](lego)
* [Navidrome](navidrome)
* [Nextcloud](nextcloud)
* [OpenLDAP](openldap)
* [PHP LDAP Admin](phpldapadmin)
* [Redmine](redmine)
* [Synapse](synapse)
* [TinyTinyRSS](ttrss)
* [Zabbix](zabbix)
* [ZNC](znc)
